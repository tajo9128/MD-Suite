# BioDockify MD Universal Documentation

Welcome to the official documentation of BioDockify MD Universal.

## Overview

BioDockify MD Universal is an autonomous molecular dynamics platform built on GROMACS, featuring:

- Adaptive multi-GPU backend (CUDA / SYCL / CPU)
- Segmented resume-safe execution
- Nanobot supervisory intelligence
- Automated publication packaging
